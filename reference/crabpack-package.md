# crabpack: Calculate Bering Sea Crab CPUE, Abundance, and Biomass

This package contains functions to calculate standard NOAA AFSC
Shellfish Assessment Program summer bottom trawl survey data products
(CPUE, Abundance, and Biomass) for the Eastern Bering Sea Shelf
(1975-present) and Northern Bering Sea Shelf (2010-present).
